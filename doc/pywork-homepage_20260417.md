# pyWork 首页模块开发

**时间**: 2026-04-17 00:37

---

## 需求

建立首页模块，要求：
- 多用户博客网站风格
- 灰色与活力绿配色
- 无图无JS模板
- 三栏显示
- 极简大气

---

## 完成内容

### 1. 首页模板

**文件**: `templates/home.html`

结构：
```
┌─────────────────────────────────────┐
│           顶部导航栏               │
├─────────┬──────────────┬────────────┤
│  左栏   │    中栏      │   右栏     │

├─────────┴──────────────┴────────────┤
```

### 2. 样式文件

**文件**: `static/css/home.css`

配色方案：
```css
灰色系: #111827 → #f9fafb (9级灰度)
活力绿: #10b981 (主色)
```

特性：
- 纯CSS，零JavaScript
- CSS Grid 三栏布局
- 响应式设计（768px 自动单栏）
- 极简卡片设计

### 3. 路由更新

**文件**: `app/main.py`

- `/` → 首页（渲染home.html）
- 自动获取最新文章填充首页

---

## 页面结构

### 左栏
- **活跃作者**: 绿色圆点标记
- **热门标签**: 标签云展示

### 中栏（主内容）
- 文章卡片列表
  - 作者 + 时间
  - 标题（可点击）
  - 摘要（160字）
  - 标签

### 右栏
- **公告卡片**: 绿色渐变背景高亮
- **统计**: 文章/用户/阅读数
- **最新评论**: 评论列表

---

## 设计特点

1. **极简主义**
   - 无装饰图形
   - 纯文字 + 线框
   - 大量留白

2. **视觉层次**
   - 灰色为主基调
   - 绿色作为强调色（按钮、链接、高亮）
   - 文字灰度区分主次

3. **三栏平衡**
   - 左右各200px
   - 中间自适应
   - 侧栏sticky定位
